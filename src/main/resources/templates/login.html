Welcome To ChoreMore!

This is a simple hero unit, a simple jumbotron-style component for calling extra attention to featured content or information.

It uses utility classes for typography and spacing to space content out within the larger container.

Parent Accounts
  • - Main account holder can create sub accounts
  • - Main account assign (CREATE) posted chores to the sub account
  • - Main account track (VIEW) posted chores and their completion of chores
  • - The main account holder then confirms (DELETE) the completion from the main account and the sub account avatar is awarded build points
  • - Main account can track posts made on the message board by sub accounts
    • Child Accounts
  • - Sub account holder creates an avatar
  • - Sub account selects a class from a pool of 5 class options, and builds (CREATE) a stat line following the scope of an RPG style game
  • - Sub accounts avatar is awarded points (UPDATE) or ‘buffs’ to build the stat line
  • - There is also a message board for sub account holders to post message and connect with friends
